# bitdefender-av Chocolatey Package

This repository contains the source files for the Chocolatey community package
**bitdefender-av** (Bitdefender Antivirus Free).

## Notes
- Installer is downloaded from the official Bitdefender website
- Silent installation is best-effort using `/silent`
- Post-install user sign-in may be required by Bitdefender

## Links
